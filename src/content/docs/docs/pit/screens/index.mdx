A reference list of every distinct UI surface in Pit. Use this to size visual work, plan Storybook stories, or trace where a feature lives. Surfaces are grouped by **shell location** — the mode or window type that hosts them.

> Storybook is the design-iteration surface for these screens. Every UI feature gets a Storybook story before functional wiring per the [Storybook-first](https://github.com/StardustMT/stardust-workspace/blob/main/CLAUDE.md) rule.
