## Why this exists

A musician's career depends on the patch loading mid-show. The dominant tool for the job is closed-source, macOS-only, and has well-known reliability gaps that any working keyboardist has stories about.

Stardust is the open-source response — built on Rust + Tauri for cross-platform support, with out-of-process plugin sandboxing so a crashing plugin doesn't take down the host. Musical theatre vocabulary (Shows, Songs, Patches, Sounds) is first-class, not bolted on. Reliability isn't a feature; it's the point.
